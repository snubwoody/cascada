A simple, high performance, general purpose UI layout engine.

## Goals

- Fast
- Simple
- Portable

## Installation
Add `cascada` to your project

```toml
[dependencies]
cascada = "0.1.0"
```

## Layout
This is a two pass layout algorithm, the minimum constraints flow up and the maximum constraints
flow down.
